Questions & Answers

What is Workflow Chain Builder?
It's a visual tool for chaining multiple API requests together. You define each step, extract data from responses, and pass variables to subsequent requests — all without writing code.
Do I need to code?
No! The interface is fully visual. You configure HTTP methods, URLs, headers, body, and transformations using simple forms. For power users, JavaScript expressions are supported for advanced transformations.
Is my data safe?
100% private. All request execution happens in your browser. No data is sent to our servers — requests go directly from your browser to the APIs you specify.
How does variable passing work?
Each step can extract values from the previous response using JSONPath expressions. These values are stored as chain variables and can be injected into headers, URL parameters, or body of subsequent steps.
Can I export my chains?
Yes! Your complete chain configuration is stored as a portable JSON file. Import it to restore your workflow at any time.
